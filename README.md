# Book Inventory Management System

This is a Book Inventory Management System built using React.

# Run the Project

 download the repository
 Open in VS Code
 Run:

npm install
npm run dev

# Open browser at:
http://localhost:5173

# feature

- Add new books
- Edit book details
- Delete books
- View book list
- Routing using React Router
